# Подготовка к собеседованиям — ProSobesim

## Цель

Подготовиться к уровню **Junior - Local (J1)** — 112 вопросов.
Платформа: [ProSobesim](https://it-incubator.io/prosobesim/react-frontend/roadmap)

---

## Стратегия подготовки

### 1. Разбей на блоки по темам

J1 React Frontend — примерные темы:

- **HTML/CSS** — базовые вопросы
- **JavaScript** — основы языка
- **React** — основы фреймворка
- **Git, HTTP, браузер** — общие знания

Группируй вопросы по темам, не учи вразнобой.

### 2. Метод «3 ведра»

На платформе есть система: **Не знаю / Средне / Хорошо знаю**. Используй её:

- Сначала пройди все 112 вопросов и честно рассортируй
- **«Не знаю»** — учи в первую очередь
- **«Средне»** — доучивай во вторую
- **«Хорошо знаю»** — просто повторяй перед собесом

### 3. Темп: 5–10 вопросов в день

- **112 вопросов ÷ 8 в день ≈ 14 дней** на первый проход
- На каждый вопрос: прочитай ответ → перескажи своими словами вслух
- **Проговаривай вслух** — на собесе нужно говорить, а не читать

### 4. Иди на собесы как можно раньше

Не жди пока выучишь всё:

- Первые 2–3 собеса будут провальными — **это нормально**
- Ты поймёшь, какие вопросы реально спрашивают
- Стресс от живого общения — отдельный навык, его надо тренировать
- После каждого собеса записывай, что не смог ответить → учи именно это

### 5. Техника ответа на собесе

Не нужно цитировать учебник. Формула:

> **Что это** (1 предложение) → **Зачем нужно** (1 предложение) → **Пример** (короткий)

---

## План на неделю

| День | Что делать                                           |
| ---- | ---------------------------------------------------- |
| 1    | Пройти все 112 вопросов, рассортировать по «вёдрам» |
| 2–4  | Учить блок «Не знаю» (по 8–10 в день)               |
| 5–6  | Учить блок «Средне»                                  |
| 7    | **Пойти на первый peer-собес** (даже если не готов)   |

Потом повторяй цикл: **собес → выявил слабые места → подучил → снова собес**.

---

## Главное правило

**Не пытайся выучить всё идеально до первого собеса.**
Практика на живых собесах даёт в 3 раза больше, чем зубрёжка.
