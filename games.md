# Варианты игр для подготовки к собесу

## Вариант 1: "Interview Arena" (RPG-квиз) ← ТЕКУЩИЙ
Ты — разработчик, сражаешься с багами-монстрами. Каждый монстр задаёт вопрос.
Правильный ответ = урон монстру, неправильный = урон тебе.
HP, XP, уровни, боссы в конце каждой темы.

**Темы-подземелья:**
- HTML/CSS → JavaScript → React → Git/HTTP/Браузер

## Вариант 2: "Кто хочет стать Junior'ом"
По аналогии с "Кто хочет стать миллионером".
Вопросы от простого к сложному, подсказки (50/50, помощь зала).
Растёт "зарплата" с каждым правильным ответом.

## Вариант 3: "Flashcard Runner"
Персонаж бежит, на пути появляются вопросы с вариантами ответов.
Правильный ответ — бежишь дальше, неправильный — теряешь жизнь.
Счётчик дистанции = прогресс.

## Вариант 4: "Квиз-дуэль с таймером"
Вопрос + таймер + очки за скорость.
Таблица рекордов, система "вёдер" (не знаю / средне / хорошо).

---

Все варианты — один HTML-файл, вопросы в JSON, легко редактировать.
